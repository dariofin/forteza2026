# SVG icon pack for reusable institution modules

This folder stores copy/paste-ready SVG source for contact and social icons.

Files:

- icon-location.svg
- icon-phone.svg
- icon-mobile.svg
- icon-web.svg
- icon-mail.svg
- icon-whatsapp.svg
- icon-facebook.svg
- icon-instagram.svg
- icon-twitter.svg

Usage flow:

1. Open one icon file.
2. Copy the full <svg>...</svg> markup.
3. Paste it inside the corresponding icon wrapper in the module HTML.
4. Keep the wrapper class names unchanged (cp-location-info\_\_icon).

Notes:

- Icons are monochrome and inherit color via currentColor.
- Keep viewBox intact to preserve sizing consistency.
